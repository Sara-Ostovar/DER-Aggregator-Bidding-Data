# Dataset for DER Aggregator Bidding Study

This repository contains the datasets used in the study:

**A Bidding Strategy Algorithm for Distributed Energy Resource (DER) Aggregators: Participation in Day-Ahead and Real-Time Energy and Ancillary Service Markets**

## Files

- `day_ahead_and_real_time_energy_prices.xlsx`  
  Day-ahead and real-time energy market price data used in the case study.

- `day_ahead_and_real_time_AS_prices.xlsx`  
  Day-ahead and real-time ancillary service price data used in the case study.

- `Solar_(PV)_Data.xlsx`  
  Solar photovoltaic (PV) data used for PV generation modeling in the case study.

## Notes

These datasets are provided to support transparency and reproducibility of the results reported in the paper.

If you use these datasets, please cite the associated paper and this GitHub repository.
